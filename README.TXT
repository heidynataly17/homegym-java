PROYECTO: HomeGym - Conexión Java con MySQL

Descripción:
Este proyecto implementa un módulo en Java utilizando JDBC para la conexión a base de datos MySQL.

Funcionalidades:
- Inserción de usuarios
- Consulta de usuarios
- Actualización de datos
- Eliminación de usuarios

Tecnologías utilizadas:
- Java
- MySQL
- JDBC
- VS Code

Instrucciones:
1. Crear la base de datos 'homegym' en MySQL
2. Crear la tabla 'users'
3. Ejecutar el archivo Main.java
